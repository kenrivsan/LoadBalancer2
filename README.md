# Tarea 07 - GraphQL Deployment

## Descripción
En esta actividad se desarrolló una API pública con GraphQL para consultar información desde una base de datos, sin necesidad de crear múltiples endpoints para cada conjunto de datos requerido.

La implementación fue realizada con Node.js, Express, Apollo Server y SQLite.
